About The Project

An API that allows users access to information about books and authors. The users can create a custom reading list of selected books. The features that the API provides are the following:

  • User authentication.
  • Query author data (pagination).
  • Query books data.
  • Reading list route authorization and CRUD operations.

Built With

  • NodeJS
  • ExpressJS

  • MongoDB

  • Description

The REST API to the example app is described below.
